# MetabolismTools

The first function just creates a bubble plot.

Install the function like this:

``install.packages("devtools")``

``install_github("mirnavazquez/MetabolismTools")``
  
``library(devtools)``

``library(ggplot2)``

Run the example:

``Bubble_plot(example,  "gene", "genomes", "abundance", "group")``
